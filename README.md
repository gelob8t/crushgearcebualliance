# Crush Gear Cebu Alliance Website

This is a static website package. Open `index.html` in a browser, or upload the entire folder to Netlify, Vercel, GitHub Pages, cPanel, or any standard web hosting.

## Included files
- `index.html` – main website page
- `styles.css` – full responsive styling
- `script.js` – mobile menu, year, and scroll animations
- `assets/logo.svg` – custom editable placeholder logo
- `assets/hero-gear.svg` – custom editable hero illustration
- `manifest.webmanifest` – basic app metadata
- `robots.txt` – search engine crawler setting

## Important notes
- Facebook access was not available during generation, so the included logo is a custom placeholder, not copied from Facebook.
- Replace `assets/logo.svg` with the official group logo if you have permission to use it.
- Replace gallery placeholders with approved photos from your group.
- Update event names, dates, and rules inside `index.html`.
